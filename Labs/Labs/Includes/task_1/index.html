Лабараторное задание №1. Условный оператор if

Условие задачи

Дана точка на плоскости с координатами (х, у). Составить программу, которая выдает одно из сообщений "Да", "Нет", "На границе" в зависимости от того, лежит ли точка внутри заштрихованной области, вне заштрихованной области или на ее границе.

Математическая модель

Уравнения, определяющие, что точка с координатами (X,Y) принадлежит заштрихованной области: x>2 и x<8, и y>3, и y<5; x>-5, и x<-3, и y>-11, и y<-1.
Не принадлежит заштрихованной области: x<2 или x>8, или y<3, или y>5; x<-5 или x>-3 или y<-11 или y>-1.
В противном случае лежит на границе.

Описание алгоритма

Для ввода точки с координатами (X,Y) описываются две переменные x и y вещественного типа float.
После ввода двух переменных x и y проверяется условие принадлежности точки к заштрихованной области выше и провее оси Ох, либо к заштрихованой области ниже и левее оси Ох.
После этого проверяется условие принадлежности точки заштрихованной области, и если это условие верно, то выводится сообщение «To4ka prenadleczit granice».
В противном случае, если это условие не верно, то проверяется условие расположения точки вне заштрихованной области, и если это условие верно, то выводится сообщение «To4ka ne prinadleczit granice».
В противном случае выводится сообщение «To4ka leczit na granice»

Блок-схема

Текст программы с комментариями

#include <iostream>
using namespace std;

int main()
{

    float x,y; //Определяем переменные
    cout<<" Wwedite zna4enie X=";
    cin>>x; //Вводим значение X
    cout<<" Wwedite zna4enie Y=";
    cin>>y; //Вводим значение Y

    if(y>0) //Условие определяющее область
    {

      if(x>2 && x<8 && y>3 && y<5) //Условие определяющее, входит ли точка в заштрихованую область
      {
        cout<<" To4ka prenadleczit granice \n";
      }
      else if(x<2 || x>8 || y<3 || y>5)
      {
        cout<<" To4ka ne prinadleczit granice \n";
      }
      else
      {
        cout<<" To4ka leczit na granice \n";
      }
    }
    else
    {
      if(x>-5 && x<-3 && y>-11 && y<-1) //Условие определяющее, входит ли точка в заштрихованую область
      {
        cout<<" To4ka prenadleczin granice \n";
      }
      else if(x<-5 || x>-3 || y<-11 || y>-1)
      {
        cout<<" To4ka ne prinadleczit granice \n";
      }
      else
      {
        cout<<" To4ka leczit na granice \n";
      }
    }

    system("pause");
    return 0;
}
