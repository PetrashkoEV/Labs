Лабораторное задание №13. Реализация класса.

Условие задачи 

    Описать класс с именем ORDER, содержащий следующие закрытые переменные – члены класса (поля):

·         расчетный счет плательщика;

·         расчетный счет получателя;

·         перечисляемая сумма в руб.

Написать программу, выполняющую ввод и вывод полей  объекта O класса ORDER.

Математическая модель

    Для решения  поставленой задачи нам необходим класс с именем ORDER (данным по условию). Этот класс содержит ф-ии типа public:  vvod_info - эта ф-ия необходима для ввода информаци с клавиатуры в поля класса типа private, и ф-ию vuvod_info -  эта ф-ия выводит поля закрытого типа класса. Определяем класс с именем ORDER, определяем ф-ии vvod_info и vuvod_info. Эти ф-ии будут описыватся далее.
    Ф-ия vvod_infoэта ф-ия необходима для ввода информаци с клавиатуры в поля класса типа private. В теле этой ф-ии будем записывать значения (счет плательщика, счет получателя, сумму снятую со счета) в переменные класса, закрытого типа.
    Ф-ия vuvod_infoэта ф-ия выводит поля закрытого типа класса. Выводим счет получателя, счет плательщика, сумму снятую со счета.                                              

Описание алгоритма

        Для решения  задачи нам необходим класс с именем ORDER (данным по условию). В этом классе будут размещены поля открытого (public) и закрытого (private) типа .  В открытом типе размещены ф-ии: vvod_info (), vuvod_info ().  Эти ф-ии обеспечивают ввод и вывод значений (счет плательщика, счет получателя, сумма) соответственно. В закрытом типе будут размешены элементы payer, эта переменная сохроняющая счет плательщика, adres, эта переменная сохроняющая счет получателя, sum, эта переменная сохроняющая сумму снятую со счета.
    В главной ф-ии определяем класс order (order o), определяем ф-ию vvod_info (o.vvod_info ()), определяем ф-ию vuvod_info (o.vuvod_info ()). Эти ф-ии будут далее рассмотрены подробно.
    Ф-ия vvod_info предназначена для ввода с клаватуры значений: счет плательщика, счет получателя, сумму.  Выводим на экран сообщения о вводе пользователем: счет плательщика, счет получателя, сумму и сохроняем эти значения в переменные payer, adres, sum  соответственно.
    Ф-ия vuvod_info предназначеа для вывода введеной информации с клавиатуры. В теле ф-ии выводм переменные payer, adres, sum, которые вводились в предедущей ф-ии.  

Текст программы с комментариями

 
#include <iostream>
using namespace std;

class order
{
public:
    void vvod_info (); // ф-ия для ввода информации с клавиатуры
    void vuvod_info (); // ф-ия для вывода информации

private:
    int payer; // переменная хронящая счет плательшика
    int adres; // переменная хронящая счет получателя
    int sum; // переменная хронящая сумму
};

void order::vvod_info() // ф-ия для ввода информации с клавиатуры
{
    cout << endl;

    cout << " Введите счет плательщика: ";
    cin >> payer ;

    cout << " Введите счет получателя: ";
    cin >> adres;

    cout << " Введите сумму: ";
    cin >> sum;
}

void order::vuvod_info () // ф-ия для вывода информации
{
    cout << endl;
    cout << " Счет плательщика:" << payer << endl;
    cout << " Счет получателя:" << adres << endl;
    cout << " Сумма:" << sum << endl;
}
int main()
{
    order o;
    o.vvod_info (); // ф-ия для ввода информации с клавиатуры
    o.vuvod_info (); // ф-ия для вывода информации

    system("pause");
    return 0;
}
