Лабораторное задание №14. Явные конструкторы и деструкторы.

Условие задачи 

    Описать класс с именем ORDER, содержащий следующие закрытые переменные – члены класса (поля):

·         расчетный счет плательщика;

·         расчетный счет получателя;

·         перечисляемая сумма в руб.

Написать программу, выполняющую ввод и вывод полей  объектов O1, O2 класса ORDER.

Математическая модель

    Для решения  поставленой задачи нам необходим класс с именем ORDER (данным по условию). Этот класс содержит конструктор, деструктор и ф-ию output, типа public:  order - эта явный конструктор который необходим для ввода информаци с клавиатуры в поля класса типа private,  ф-ия output  выводит поля закрытого типа класса, и деструкор выводит сообщение о том что объект уничтожен. Определяем класс с именем ORDER, определяем конструктор order, деструктор ~order и ф-ию output. Они будут описыватся далее.
    Явный конструктор  order  необходим для ввода информаци с клавиатуры в поля класса типа private. В теле явного конструктора будем записывать значения (счет плательщика, счет получателя, сумму снятую со счета) в переменные класса, закрытого типа.
    Ф-ия output эта ф-ия выводит поля закрытого типа класса. Выводим счет получателя, счет плательщика, сумму снятую со счета.
    Явный деструктор ~order выводит соодщение о том что объект уничтожен.                                                 

Описание алгоритма

        Для решения  задачи нам необходим класс с именем ORDER (данным по условию). В этом классе будут размещены поля открытого (public) и закрытого (private) типа .  В открытом типе размещены: вный конструктор, order (), явный деструктор, ~order (), и ф-ия: output ().  Они обеспечивают ввод и вывод значений (счет плательщика, счет получателя, сумма) соответственно. В закрытом типе будут размешены элементы payer, эта переменная сохроняющая счет плательщика, adres, эта переменная сохроняющая счет получателя, sum, эта переменная сохроняющая сумму снятую со счета.
    В главной ф-ии определяем класс order (order o1), определяем ф-ию output () (o.output ()), определяем явный деструктор  ~order () (o1.~order ()).  Эти ф-ии будут далее рассмотрены подробно. Явный конструкорв главной ф-ии  не определяем так как он определяется автоматически.
    Явный конструктор order() предназначен для ввода с клаватуры значений: счет плательщика, счет получателя, сумму.  Выводим на экран сообщения о том что обьект создан и о вводе пользователем: счет плательщика, счет получателя, сумму и сохроняем эти значения в переменные payer, adres, sum  соответственно.
    Ф-ия output () предназначеа для вывода введеной информации с клавиатуры. В теле ф-ии выводм переменные payer, adres, sum, которые вводились в предедущей ф-ии. Эта ф-ия определена как константа (void output () const) т.е. она только выводит данные и ничего не меняет в них. Это необходимо для избежания ошибок на этапе компиляции.
    Явный деструктор  ~order () выводит сообщение о том что объект уничтожен.

Текст программы с комментариями

 
#include <iostream>
using namespace std;

class order
{
public:
    order(); // явный конструктор
    void output () const; // ф-ия вывода счетов и суммы, которая определена как константа.
    ~order (); // явный деструктор
private:
    int payer; // счет плательщика
    int adres; // счет получателя
    int sum; // сумма
};

order::order() // явный конструктор
{
    cout << " Объект создан " << endl; // сообщение о том что объект создан

    cout << " Введите счет плательщика: "; // сообщение о том что необходимо ввести счет плательщика
    cin >> payer;

    cout << " Введите счет получателя: "; // сообщение о том что необходимо ввести счет получателя
    cin >> adres;

    cout << " Введите перечисляемую сумму: "; // сообщение о том что необходимо ввести перечисляемую сумму
    cin >> sum;
}

void order::output () const // ф-ия вывода счетов и суммы, которая определена как константа.
{
    вывод данных
    cout << endl;
    cout << " Счет плательщика: " << payer << endl;
    cout << " Счет получателя: " << adres << endl;
    cout << " Сумма: " << sum << endl;
}

order::~order () // явный деструктор
{
    cout << " Обьект унечтожен !!!" << endl; // вывод сообщения о том что объекты класса уничтожены
}

int main()
{
    order o1;
    o1.output (); // определение ф-ии вывода счетов и суммы, которая определена как константа. 
    o1.~order (); // определение явного деструктора

    system("PAUSE");
    return 0;
}
