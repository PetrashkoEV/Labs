Лабораторное задание №15. 

Класс для обработки двумерного целочисленного массива.

Условие задачи 

    Написать программу, в которой предусмотреть класс для организации обработки двумерного целочисленного массива. Внутри класса реализовать два конструктора: 1–й конструктор должен обеспечивать ввод элементов массива с клавиатуры; 2—й конструктор должен формировать значения элементов массива в задаваемом диапазоне [min, max] с помощью генератора случайных значений. Внутри класса предусмотреть следующие методы:

·        вывод массива на экран;

·        вывод в каждой строке сначала всех отрицательных элементов, потом всех остальных;

·    вычисление значения  суммы элементов в тех строках, которые содержат хотя бы один отрицательный элемент;

·           вывод номеров строк и столбцов всех седловых точек матрицы (Матрица А имеет седловую точку Аij, если Аij является минимальным элементом в i–й строке и максимальным в j–м столбце).

                                          

Описание алгоритма

        Для решения  задачи нам необходим клас int_mass.  В этом классе будут размещены поля открытого (public) и закрытого (private) типа .  В открытом типе размещены ф-ии: inpt_1_mass, предназначеной для формирования массива вводом чисел с клаватуры, inpt_2_mass, ф-ия предназначена для формирования массива при помощи ГСЧ, output_mass, метод по выводу сформулированого массива на экран, outut_negativ, метод по выводу сначала всех отрицательных, а потом всех остальных элементов, sum_mass, метод по вычислению суммы элементов в тех строках , которые содержат  хотя бы один отрицательный элемент, saddle, ф-ия по нахождению седловых точек. В закрытом типе будут размешены: двумерный массив(mass[10][10]), минимальное (min) и максимальное (max) число в массиве сформированого ГСЧ.
        В главной ф-ии выведем на экран сообщение о том что необходимо ввести размерность массива. Определим "флаг" (bool flag), зачем он нужен будет описано поздней. Затем решение задачи разобьется на два вида массива: для значений введеных с клавиатуры и значений сформированых ГСЧ в определенном промежутке.
        Мы рассмотрим решение тольоко для ввода чисел с клавиатуры. т.к. эти методы будут использовать одинаковые ф-ии. Различие будет состоять в том что при вводе чисел с клавиатеры мы будем использовать ф-ию inpt_1_mass, а при формирований чисел ГСЧ используется ф-ия  inpt_2_mass.
        Ввод чисел с клавиатуры: для начала необходимо определить массив (int_mass M1) и ввести данные в массив при помощи ф-ии M1.inpt_1_mass. Ввывести сообщение о том что массив был сформирован и вывести его на экан при помощи ф-ии output_mass (M1.output_mass (n)). Затем подключить ф-ию outut_negativ, по выводу сначала всех отрицательных а потом всех остальных элиментов, и подключить ф-ию sum_mass, по вычислению значения суммы эл в тех строках которые содержат хотя бы один отрицательный элемент. Присвоить "флагу" значение false (flag=false). Определить цикл for (for (int k=0; k<n; k++)) для того чтобы на вход ф-ии подовался двумерный массив построчно. Определим ф-ию saddle., по вычислению седловых точек в массиве. т.е. если матрица А имеет седловую точку Аij, если Аij является минимальным элементом в i–й строке и максимальным в j–м столбце. Все эти ф-ии ,кроме saddle, на вход получают размерность, а ф-ия saddle на вход получает размерность массива, номер строки которая будет обрабатывятся и по ссылке передается "флаг". Флаг предназначен для того чтобы вывести сообщение о том что в массиве нет седловых точек, если действительно таких точек нет. т.к. когда хотябы одна точка найдется то флагу будет присвоено значение true. Все ф-ии будут подробно описаны далее.
    Ф-ия inpt_1_mass. Предназначена для формирования массива вводом чисел с клавиатуры. Определим два цикла for , предназначеных для вычисления номера строки и номера столбца, и выведем сообщение о том что необходимо ввести Аij элимент.
    Ф-ия inpt_2_mass. Предназначенной для формирования массива при помощи ГСЧ. Выведем сообщение о том что необходимо ввести max и min значения. определить ГСЧ (srand (( unsigned) time(NULL))) и также определим два цикла for, предназначеных для вычисления номера строки и номера столбца, и присвоим некоторому элименту сформированое ГСЧ значение в определенном промежутке (mass[i][j]=rand()%(max-min+1)+min).
        Ф-ия output_mass. Ф-ия по выводу сформулированого массива на экран. Снова определим два цикла for, предназначеных для вычисления номера строки и номера столбца, и выведем на экран элименты.
        Ф-ия outut_negativ. Метод по выводу сначала всех отрицательных, а потом всех остальных элементов. Выведем сообщени о том что сейчас будет выведен отсортированый массив. Определим 1-ый цикл for (for (int i=0; i<n; i++)), цикл по вычисления номера строки. В теле этого цикла опрнделим еще 2 цикла for: 1-ый для вывода отрицательных чисел, а 2-ой для вывода всех остальных элиментов. Определив уже 3-й цикл for (for (int j=0; j<n; j++)), для вывода отрицательных чисел. Определим одну условную конструкцию if (mass[i][j]<0) которая будет проверять элимен на отрицательность. Если элимент отрицательный то его необходимо вывести на экран. В 3-ем цикле for (for (int j=0; j<n; j++)), для вывода всех остальных элементов. Определим одну условную конструкцию if (if (mass[i][j]>=0)) которая будет проверять элимент на положительность и выводить его на экран. т.о. после этих 3 циклов на экан будет выведен отсортированый массив в порядке возростания.
        Ф-ия sum_mass. метод по вычислению суммы элементов в тех строках , которые содержат  хотя бы один отрицательный элемент. Выведем на экран сообщение что сейчас будет выведена сумма строки в которой присутствует хотябы один отрицательный элимент и сделаем небольшую заготовку таблицы. Определим цикл for предназначеный для вычисления некоторго ромера строки. В теле этого цикла определим переменную summ (int summ=0) котороя будет хранить сумму элементов строки и предварительно занулим эту переменную. Это сделано для того что если в строке нет отрицательных элиментов то вместо значения суммы выведется значение 0. Далее определим цикл for который вычисляет номер некоторого элемента в i строке. А в теле этого цикла определим условную конструкцию if (if (mass[i][j]<0)) которая будет проверять элименты на отрицательность. Если элимент отрицательный то в нутри конструкции определим цикл for предназначеный для вычисления номера столбца в i строке с начала. А в теле этого цикла к переменной summ будем добовлять некоторый элемент строки mass[i][k] (summ+=mass[i][k]). т.е. быдет происходить вычисление суммы строки. Как только подсчет суммы завершен то выполнится команда break по выходу из 2-го цикла и для перехода в новую строку. Это сделано на тот случай если в строке несколько отрицательных элимента. Но в конце 1-го циква произойдет вывод: номер строки и сумма в этой строке в виде таблици.
        Ф-ия saddle. Это ф-ия по нахождению седловых точек. Для начала необходимо определить переменую index равную 0 (int index=0). Эта переменная будет хранить номер столбца  маскимального элемента. Далее будем находить максимальную точку в строке. Определим переменную max равную mass[i][0], 1-му элементу в i-строке. Далее определим цикл for для вычисления номера солбца в некоторой строке. Условная конструкция if (if (mass[i][j]<min)) будет проверять элементы строки на min. Если элемент больше min, где переменную min (int min=mass[i][0]) мы определли первым элементом, то в переменную index сохроняется номер столбца максимального элемента. и переменной max присваиваем зачение большей переменной (index=j). Как только цикл закончился то мы нашли максимальный элемент в строке и зафиксировали номер столбца этого элемента. теперь мы будем находить минимальный элемент в этом столбце. Но для начала определим цикл (for (int j=0; j<n; j++)) и произведем проверку if (), на тот случай если в строке несколько одинаковых минимальных элементов. После чего определим переменную max и присвоем ей значение первого элемента столбца в котором мы нашли максимальный элемент (int max=mass[0][index];). Далее определим цикл  for (for (int k=0; k<n; k++)) для вычисления номера строки фиксированого столбца. И опять создадим условную конструкцию if ( if (mass[k][index]>max)) которой будем проверять на максимум все элементы солбца. В теле условной конструкции мы переменной max присвоим значение минимального элемента столбца (max=mass[k][index]) и переменной stroka присвоим номер строки в которой найден минимальный элемент (stroka=j). После выхода из цикла мы определили минимальный элемент в фиксированой строке. Теперь определим условную конструкцию if (if (max==min)) и проверим максимальный и минимальный элементы. Это сделано на тот случай если в строке окажется несколько одинаковых элементов совпадающих по значению максимальных элементов в i строке. Если эти два критерия совпали то "флагу" присваиваем значение true (flag=true) , и  на экран выводится сообщение о том что седловая точка найдена и выведем на экран координаты этой точки.

# include <iostream>
# include <time.h>
using namespace std;

class int_mass
{
public:
    inpt_mass (int n); // формирование массива вводом чисел с клавиатуры
    inpt_mass (int n); // формирование массива при помощи ГСЧ
    void output_mass (int n) const; // метод по выводу сформульрованого массива на экран
    void outut_negativ (int n) const; // метод по выводу сначала всех отрицательных элементов, потом всех остальных;
    void sum_mass (int n); // вычисление значения суммы элементов в тех строках, которые содержат хотя бы один отрицательный элемент
    void saddle (int n, int i, bool &flag); // нахождение седловых точек
private:
    int mass[10][10]; // заданый массив
    int min; // минимальное число в массиве сформированого при помощи ГСЧ
    int max; // максимальное число в массиве сформированого при помощи ГСЧ
};

void int_mass::inpt_1_mass(int n)
{
    for (int i=0; i<n; i++)
    {
        for (int j=0; j<n; j++)
        {
            cout << " Введите A " << i+1 << ',' << j+1 << ": ";
            cin >> mass[i][j];
        }
        cout << endl;
    }
}

void int_mass::inpt_2_mass (int n) // конструктор создающий массив при помощи ГСЧ в заданом деопазоне
{
    cout << " Введите min значение: "; // ввод минимального числа в массиве
    cin >> min;

    cout << " Введите max значение: "; // ввод максимального числа в массиве
    cin >> max;

    cout << endl;

    srand (( unsigned) time(NULL)); // определение ГСЧ
    for (int i=0; i<n; i++)
    {
        for (int j=0; j<n; j++)
        {
            mass[i][j]=rand()%(max-min+1)+min; // формирование числа
        }
    }
}

void int_mass::output_mass (int n) const // вывод сформульрованого массива на экран
{
    for (int i=0; i<n; i++)
    {
        for (int j=0; j<n; j++)
        {
            cout << mass[i][j] << "    ";
        }
        cout << endl;
    }
    cout << endl;
}

void int_mass::outut_negativ (int n) const // метод по выводу сначала всех отрицательных элементов, потом всех остальных;
{
    cout << "Ввывод массива где сначала идут отрицательные, а затем положительные элименты: " << endl;
    cout << "________________________" << endl;
    for (int i=0; i<n; i++)
    {
        // вывод отрицательных чисел
        for (int j=0; j<n; j++)
        {
            if (mass[i][j]<0)//
            {
                cout << mass[i][j] << "  ||  "; // вывод отрицательных чисел в строке
            }
        }

        //вывод положительных чисел
        for (int j=0; j<n; j++)
        {
            if (mass[i][j]>=0)//
            {
                cout << mass[i][j] << "  ||  "; // вывод положительных чисел в строке
            }
        }
        cout << endl;
    }
    cout << "________________________" << endl;
    cout << endl;
}

void int_mass::sum_mass (int n) //вычисление значения суммы элементов в тех строках, которые содержат хотя бы один отрицательный эл
{
    cout << " Вычисление суммы в тех строках в которых есть хотя бы один отрицательный элемент " << endl;
    cout << "№ стр|| сумма" << endl;
    for (int i=0; i<n; i++)
    {
        int summ=0;
        for (int j=0; j<n; j++)
        {
            if (mass[i][j]<0)
            {
                for (int k=0; k<n; k++)
                {
                    summ+=mass[i][k]; //вычисление значения суммы элементов
                }
                break;
            }
        }
        cout << "  " << i+1 << "  || " << summ << endl;
    }
}

void int_mass::saddle (int n, int i, bool &flag) // вычисление седловых точек
{
    int index=0; // переменная хранящая номер столбца максимального элемента

    // нахождение максимальной точки в строке

    int min=mass[i][0]; // переменная хоанящая значение max элемента в i строке
    for (int j=0; j<n; j++)
    {
        if (mass[i][j]<min) // проверка элемента строки на максимум
        {
            index=j; // сохранения номера столбца максимального элемента
            min=mass[i][j]; // присвоение значения максимального элемента в строке
        }
    }
    for (int j=0; j<n; j++)
    {
        if (min==mass[i][j])
        {
            int max=mass[0][j]; // переменная хранящая минимальное значение в j строке

            for (int k=0; k<n; k++)
            {
                if (mass[k][j]>max) // проверка элемента строки на минимум
                {
                    max=mass[k][j]; // присвоение значение минимального  элемента в столбце
                }
            }
            if (max==min) // если элеенты максимума и минимума
            {
                cout << " Седловая точка с координатами [" << i+1<< ", " << j+1 << "]" << endl;
                flag=true;
            }
        }
    }
}

        cout << " Сформированый при помощи ГСЧ массив: \n" << endl;

        M2.output_mass (n); // метод по выводу сформульрованого при помощи ГСЧ массива на экран
        M2.outut_negativ (n); // метод по выводу сначала всех отрицательных элементов, потом всех остальных
