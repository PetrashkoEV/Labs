Лабораторное задание №2. Оператор SWITCH

Условие задачи

Единицы длины пронумерованы следующим образом: 1-дециметр, 2-километр, 3-метр, 4-миллиметр, 5-сантиметр. Дан номер единицы длины и длина отрезка L в этих единицах (вещественное число). Вывести длину данного отрезка в метрах.

Математическая модель

Для решения задачи будем использовать разветвление процесс вычислений на 6 направлений:
1.
Если пользователь выбирает "1-decimetr" (1 направление), то переводим длинну отрезка L из децеметров в метры. Для этого используем формулу: X=L/10, и выводим результат на экран.
2.
Если пользователь выбирает "2-kilometr" (2 направление),то переводим длинну отрезка L из километров в метры. Для этого используем формулу: x=L*1000 и выводим результат на экран.
3.
Если пользователь выбирает "3-metr" (3 направление),то переводим длинну отрезка L из метров в метры. Для этого используем формулу:X==X и выводим результат на экран.
4.
Если пользователь выбирает "4-millimetr" (4 направление),то переводим длинну отрезка L из миллиметров в метры. Для этого используем формулу: X=L/1000 и выводим результат на экран.
5.
Если пользователь выбирает "5-santimetr" (5 направление),то переводим длинну отрезка L из сантиметров в метры. Для этого используем формулу: X=L/100 и выводим результат на экран.
6.
Если пользователь выбирает колонку с номером больше чем пять, то на экран выводится "Debil!!! Nuzno wubirat' iz spiska (sm. spisok wuhe)" и следует выбрать колонку с номером меньше пяти.

Описание алгоритма

Для вычисления длины отрезка L из децеметров, километров, метров, миллиметров, сантиметров в метры описываются три переменные nd,L,x вешественного типа int.
После ввода номера едениц измерения и длины отрезка L будем использовать оператор switch, предназначеный для разветвления процесса вычислений на 6направлений.
Затем будем использовать заранее заданое направление case(1-5). Далее будет происходить вычисление длины отрезка L по формулам представленным в "Математической модели". После того как выполнится вычесление длины на экран выводится сообщение " Dlinna otrezka L w metrax = ".
Если же будет выбрано направление case больше пяти, то на экран выводится сообщение " Debil!!! Nuzno wubirat' iz spiska (sm. spisok wuhe)".

Блок-схема

Текст программы с комментариями

#include<iostream>
#include <conio.h> //для использования ф-и getch
using namespace std;

int main()
{

    double nd,L,x; //Определяем переменные
    int nd; //Определяем переменные
    cout<<" 1-decimetr \n";
    cout<<" 2-kilometr \n";
    cout<<" 3-metr \n";
    cout<<" 4-milimetr \n";
    cout<<" 5-santimetr \n";
    cout<<" Wwedite nomer edenicu dlinnu"; //Вводим номер еденицы строки
    cin>>nd;
    cout<<" Wwedite dlinnu otrezka L. L= "; //Вводим длину отрезка.
    cin>>L;
    switch(nd) // номер еденицы длины
    {
    case 1: x=L/10;
      cout <<" Dlina otrezka L w metrax = "<< x << endl;
      break;
    case 2: x=L*1000;
      cout <<" Dlina otrezka L w metrax = "<< x << endl;
      break;
    case 3: x=L;
      cout <<" Dlina otrezka L w metrax = "<< x << endl;
      break;
    case 4: x=L/1000;
      cout <<" Dlina otrezka L w metrax = "<< x << endl;
      break;
    case 5: x=L/100;
      cout <<" Dlina otrezka L w metrax = "<< x << endl;
      break;
    default: cout <<" Debil!!! Nuzno wubrat' iz spiska (sm. spisok wuhe)";
    }
cout << endl; //Перевод курсора на следующую строку.
system("pause");
return 0; //Окончание операции.
}
