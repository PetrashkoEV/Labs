Лабораторное задание №6. Функции.

Условие задачи


    Написать функцию Cos1(x,eps) вещественного типа (параметры x, eps — вещественные, eps > 0), находящую приближенное значение функции cos(x), где cos(x) = 1 – x2 / 2! + x4 / 4! – ... + (–1)n x2n / (2n)! + ... В сумме учитывать все слагаемые, большие по модулю eps.

Математическая модель

Для нахождения приближённого значения функции Cos1() введём формулу Cos1(x)= 1 – x2 / 2! + x4 / 4! – ... + (–1)n x2n / (2n)! + ...
В сумме также будем учитывать все слагаемые, большие eps.

Описание алгоритма

Для решения этой задачи введем функцию Cos1(). Которая будет вычислять 1 – x2 / 2! + x4 / 4! – ... + (–1)n x2n / (2n)! + ... . В функции Cos1() задатим две фактические переменные вещественного типа x и eps. В цикле будем использовать переменные у, g, s.(у=-1, g=1, s=1). Переменная "у" это переменная вычисляющая знак. Переменная "g" это переменная которая вычисляет x2n / (2n)!, а переменная "s" это сама переменная вычисляющая значение Cos при помощи переменных у, g, х и n. N-это счетчик цикла for, который будет прибавлять 1 и цикл будет выполнятся до тех пор пока переменная g  будет больше или равно eps, которое задается пользователем.  

После составления функции пишем саму программу, в которой сразу же определяем две переменные типа x и epsilon, значения которым будет задавать пользователь программы, вводя значения с клавиатуры.
Теперь просто выведем на экран значение от выполнения функции Сos1(), т.е. после ввода данных с клавиатуры, они передадутся в функцию cos1(), которая обработает эти данные и вернёт соответствующее значение.

 

Блок-схема

Текст программы с комментариями

#include <iostream>
using namespace std;
double cos1(float x, float eps)
{   
    int y=1;
    float g=1;
    float s=1;
    for (int n=1; g>=eps; n++)
    {
        y*= -1; // знак
        g*=x*x/(2*n)/(2*n-1);
        s+=y*g;
    }
    return s;
}
int main ()
{
    float x,eps;
    float z;
    cout << " Wwedite zna4enie cosinusa uygla, x:";
    cin >> x;
    cout << " Wwedite to4nost' wupolneniia ";
    cin >> eps;
    z=cos1(x,eps);
    cout << " Zna4enie cos("<< x <<")=" << z << endl;
    system ("pause");
    return 0;
}
