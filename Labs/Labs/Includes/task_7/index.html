Лабораторное задание №7. Вызов функции по ссылке.

Условие задачи


Написать функцию Cos1(x,eps) вещественного типа (параметры x, eps — вещественные, eps > 0), находящую приближенное значение функции cos(x), число n и величину последнего элемента, где cos(x) = 1 – x2 / 2! + x4 / 4! – ... + (–1)n x2n / (2n)! + ... В сумме учитывать все слагаемые, большие по модулю eps.

Математическая модель

    Для нахождения приближённого значения функции Cos() введём формулу Cos(x)= 1 – x2 / 2! + x4 / 4! – ... + (–1)n x2n / (2n)! + ... , найдем величину n и значение последнего элемента.
    В сумме также будем учитывать все слагаемые, большие eps.

Описание алгоритма

    Для решения этой задачи введем функцию Cos(). Которая будет вычислять косинус угла по фомуле : 1 – x2 / 2! + x4 / 4! – ... + (–1)n x2n / (2n)! + ... номер последнего элемента (num) и его значение (last). В теле этой ф-ии cos определим переменные преденазначеные для:  вычисление значения знака в зависемости от n (y=1), вычисления значения последнего члена (next=1), вычисление велечины угла (s=0). И присвоение переменной, по вычислению последнего значения, значение равное 0 (last=0).  Далее определим цикл (for (num=0; next>=eps; num++) , где num сохроняет номер последнего элемента), а в теле цикла присвоим переменной last значение next (last=next), где last это значение последнего элемента, по умолчянию равно 1, а next это велечина следующего члена ряда. А к переменной s ,вычисляющей значение косинуса угла, прибовляем "знак" (s+=y*last). затем переменную вычисляющую знака умножаем на (-1). И по формуле: 1 – x2 / 2! + x4 / 4! – ... + (–1)n x2n / (2n)! + ... вычисляем значение текущего элемента элемента (next*=x*x/(2*num+1)/(2*num+2);). После завершения цикла в переменной num хронится номер последнего элемента в переменной last хранится велечина этого элемента, где эти две переменные выводим на экран, а в переменной s хранится значения косинуса в данном угле.

    В главной ф-ии определим переменные x,eps,last, num, где x это угол введеный с клавиатуры пользованелем, eps это точность выполнения которая также вводится с клавиатуры, last это переменная хранящая велечину последнего элемента, а num это переменная вычисляющая номер последнего элемента. Затем после определения этих элементов просим пользователя ввести значение угла и точность выполнения. Затем определим ф-ию cos, описаную выше, и выведем на экран велечину косинуса угла.

Блок-схема

Текст программы с комментариями


#include <iostream>
using namespace std;

double cos(double x, double eps, double &last, int &num)
{  
    int y=1;
    double next=1, s=0;
    last=0;
    for (num=0; next>=eps; num++)
    {
        last=next; // вычисление последнего члена
        s+=y*last; // вычисление косинуса уга
        y*= -1; // вычисление значение знака в зависимисти от n
        next*=x*x/(2*num+1)/(2*num+2); // вычисление значение n-го элемента бес учета знака
    }
    cout << " Значение n="<< num << endl;
    cout << " Величина последнего элемента: " << last <<endl;
    return s;
}
int main ()
{
    double x,eps,last;
    int num;

    cout << " Введите угл косинуса x: ";
    cin >> x;

    cout << " Введите точность выполнения: ";
    cin >> eps;

    cout << " Значение cos(" << x << ")=" << cos(x, eps, last, num) << endl;
    system ("pause");
    return 0;
}
